第二回レポート

VineLinux6.3インストールの手順

1.インストールディスクをドライブに入れ、起動する。

モノによってはBIOSの設定画面を開き、起動を試みるハードの順序を変える必要があるが、ここではそのまま起動できたので省く。

2.ディスクからインストールウィザードが立ち上がる。

「次へ」をクリック

3.言語の設定

そのまま「日本語」を選択して「次へ」

4.キーボード設定

「日本語」を選択して「次へ」

5.インストールの種類

すべて このオプションを選択するとGNOMEデスクトップやサーバソフトウェアを含む全てのソフトウェアパッケージがインストールされます。
デスクトップ パーソナルコンピュータやラップトップ用に、一般的なグラフィカルデスクトップ環境とホームあるいはデスクトップ使用に理想的なシステムを構成するにはこのインストールタイプを選択します。
サーバ ファイル共有、印刷共有及びウェブサービスを設定したい場合、このインストールタイプを選択します。追加のサービスも有効にできて、グラフィカル環境をインストールするかどうかを選択できます。
ベースシステム 最小構成のシステムを構成したい場合は、このインストールタイプを選択してください。
カスタム ソフトウェアパッケージの選択や、パーティションの設定を含めたインストールプロセスを完全にコントロールしたい場合に、このインストールタイプを選択してください。

どれを選ぶか少し迷ったが、無難に「すべて」を選ぶことにした。

6.ディスクパーティションの設定

とくにこだわりはないので「自動パーティション設定」を選択して「次へ」「初期化が必要」との警告に「はい」

7.自動パーティション設定

元々のHDDにはLinux以外のパーティションは無いはずなので「システムのすべてのLinuxパーティションを削除」を選択。「次へ」

8.ブートローダの設定

第12回目でGRUBを使うようなのでそのままインストールする

9.ネットワークの設定

「編集」をクリックし、出てきたダイアログボックスの「DHCPを使用して設定」のチェックボックスを外して割り当てられたように入力する。

ホスト名も「jisshuu2」として、ゲートウェイ・DNSも入力する。

10.ファイアウォール設定

「ファイアウォールを有効にする」のラジオボタンをクリック。

先の実習を見越して、FTP以外のSSH,HTTP,SMTPへのアクセスを許可する。「次へ」。

11.追加の言語サポート

そのまま「次へ」

12.タイムゾーンの選択

国際標準時との差異を設定するものなのでヤクーツクを選んでも良いが無難に「アジア/東京」を選択する

13.Rootパスワードの設定および一般ユーザアカウントの作成

当然だがRootパスワードは一般ユーザのパスワードと違うものにする。

任意のユーザ名・フルネームとそれぞれ違ったパスワードを入力して「次へ」

14.インストールパッケージのデフォルト設定

あとであれが無かった、これが無かった、となるのは嫌なのでオススメされた通りに「次へ」

15.インストール準備完了

「次へ」を押して30分ほど待てばインストールが完了する。

ディスクを取り出しもう一度電源ボタンを押せばVineLinuxが起動する。

パッケージ一覧、システムの更新

apt-get
パッケージ(プログラム)をインストール/アンインストール/アップデートするコマンド。それぞれのパッケージの操作がひとつのコマンドで行える。
$ sudo apt-get update

まずはパッケージのリストを最新のものに更新して、

$ sudo apt-get upgrade

更新があるパッケージをインストールする。

X Window System,音源の有効化

X Window Systemはインストールの種類「すべて」のなかに含まれており、問題もなかったので無事に立ち上がった。

音源も元からインストールされていたドライバで有効になっていた。
